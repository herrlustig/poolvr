poolvr

Play pool / billiards in WebVR with your Leap Motion or HTC Vive controllers!


Play now! (development version 0.1.1)


Ask questions / report issues / give feedback / fork the code / join the development team - at the poolvr GitHub repository!


Obtaining the latest WebVR-enabled browsers:

  • Mozilla Firefox supports WebVR in its nightly builds, see MozVR for more info
  • WebVR-enabled builds of Chrome / Chromium
