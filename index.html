poolvr

Play pool / billiards in WebVR with your Leap Motion or HTC Vive controllers!

poolvr is in perpetual development - you can try a current version here!

Ask questions / report issues / give feedback / fork the code / join the development team - at the poolvr GitHub repository!


poolvr videos on YouTube

Obtaining the latest WebVR-enabled browsers:

Playing in VR using Oculus Rift or HTC Vive currently requires a recent (supporting WebVR 1.0 draft spec) version of WebVR-enabled Chromium, which can be found at webvr.info.
